NXcustom ToolBars                                 Last Modified: 8Jan2020

Use the ToolBars folder to share modifications to NX tool bars and ribbons with multiple users.

The following line in the NXcustom\NX1899library\NX_env.dat file controls the location and folder name of the ToolBars folder:

NXCUSTOM_TOOL_BARS=${NXCUSTOM_LIB}\ToolBars\

Within this folder you will find the some sample modifications to some tool bars and the NXcustom Ribbon Tab.
